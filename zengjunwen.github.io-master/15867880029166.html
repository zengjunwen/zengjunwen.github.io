Spring AOP
$[timeformat('2020-04-13T22:26:42+08:00')]

Aspect-Oriented Programming (AOP) complements Object-Oriented Programming (OOP) by providing another way of thinking about program structure. AOP实现了业务逻辑与日志（Tracing）、异常处理（Exception）、事物（Transactions）、安全（Security）解耦，更加的聚焦于业务逻辑的编写。

Spring AOP的设计模式采用了代理模式，代理模式分为静态代理、JDK动态代理接口、CGlib代理类。静态代理比较简单，不进行描述。JDK动态代理只能对接口进行代理，如果要对类进行代理只能使用CGlib进行代理。

JDK动态代理基于反射实现；CGLIB包的底层是通过使用一个小而快的字节码处理框架ASM，来转换字节码并生成新的类。
