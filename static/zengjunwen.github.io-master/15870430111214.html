Java类加载
$[timeformat('2020-04-16T21:16:51+08:00')]

类加载分为几个阶段完成，装载、链接（验证、准备、解析）、初始化。

装载

将字节流转化为方法区的数据结构，堆中生成代表该类的java.lang.Class对象，提供给方法区的访问入口。

准备

为静态变量在方法区分配内存，初始化为默认值。

解析

符号引用转为直接引用。

初始化

在发生以下动作时触发初始化：

  1. new
  2. 访问静态方法、变量
  3. 反射
  4. 初始化子类

内存

方法区

加载的类信息、常量、静态变量、方法代码。

常量池

方法区的一部分，存放常量和类中的符号引用。

堆区

存放类的对象实例。

栈区

Java虚拟机栈，由一个个栈帧组成的后进先出的栈式结构。栈帧中存放方法运行时产生的局部变量、方法出口等信息。当调用一个方法时，虚拟机栈中会创建一个栈帧存放这些数据，发放调用完成时，栈帧消失，如果方法中调用了其他方法，则继续在栈顶创建新的栈帧。
