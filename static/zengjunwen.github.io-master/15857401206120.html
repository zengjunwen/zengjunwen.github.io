Bagging

个体学习器间不存在依赖关系，模型并行学习。关注方差。

结合策略(combine)

  1. Averageing
    Hard Voting：平均投票。
    Soft Voting：加权平均投票。
  2. Stacking
  3. 所谓Stacking 即，Model stacking: Train additional model to combine individual learners。

Boosting

串行成序列化方法，关注损失函数，关注偏差。

AdaBoost（adaptive boosting）实质上是基于加性(additive)模型以类似牛顿迭代法来优化指数损失函数，该模型使得对先前基学习器做错的训练样本在后续受到更多的关注，然后基于调整后的样本分布来训练下一个基学习器。

Gradient Boosting对残差(负梯度)进行拟合。
